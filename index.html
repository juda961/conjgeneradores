Verificador de Conjunto Generador

Notas:
  • Escribe cada vector en una línea. Componentes separadas por coma o espacio.
  • Ejemplo (3 vectores en R³):
    1, 0, 2
    0, 1, -1
    2, 1, 3
  • El programa arma una matriz con esos vectores como columnas. La dimensión del espacio objetivo se toma como el número de componentes (número de columnas).
