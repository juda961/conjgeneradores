Verificador de Conjunto Generador

Notas:
  • Escribe cada vector en una línea. Componentes separadas por coma o espacio.
  • Ejemplo (3 vectores en R³):
    1, 0, 2
    0, 1, -1
    2, 1, 3
  • El programa arma una matriz con esos vectores como columnas. La dimensión del espacio objetivo es el número de componentes.

Entrada

Diseñado para comprobar si los vectores generan \(\mathbb{R}^m\).
